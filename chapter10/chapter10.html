Usando links para download

Clique aqui para baixar esse PDF.

<a> href="10 - Ligações em toda parte.pdf" download="ligacoes.pdf" type="application/pdf">PDF</a>

O parâmetro download define o hiperlink como um caminho para baixar o arquivo localizado em href. Você pode renomear o arquivo e alterar sua extensão dessa forma: download="novonome"

Esse site explica bem como funciona as tags para download de arquivos.
