Imagens Dinâmicas

As imagens de um site não devem ser muito grandes pois podme prejudicar a experiência do usuário e tornar a navegação muito pesada, caudando lentidão. Porém, as imagens devem se adaptar ao tamanho da tela que o usuário utiliza.

Nesse capítulo vamos aprender a redimensionar a imagem com base no tamanho da tela do usuário.

Aqui embaixo temos 3 imagens que vão se alternando conforme você visualiza de um dispositivo diferente ou aumenta/diminui o tamanho da janela do seu navegador:

Para isso acontecer, usamos a tag picture e, dentro dela, as tags source:media para definir os parâmetros que transformam a imagem e img que traz a imagem padrão.

A ordem das tags source é essencial! Precisamos priorizar as que definem o menor tamanho.
