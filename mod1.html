Capítulo 9: Listas

Trabalhando com Listas

Listas Ordenadas

As listas ordenadas iniciam com a tag ol seguido do parâmetro type, que é onde definimos o tipo de lista que queremos (letras, números , números romanos, etc). Podemos também utilizar o parâmetro start, usado para definir de qual número/letra a lista deve iniciar.

  1. Essa é uma lista
  2. Ordenada por números
  3. Cada item deve iniciar com a tag li
  4. Dentro do HTML5 não é obrigado o fechamento da tag
  5. Então basta se preocupar em abrir a tag
  6. Mas o fechamento da tag ol é essencial

Listas não-ordenadas

O nome é auto-explicativo. Usamos a tag ul para montar essas listas e o parâmetro type para definir o tipo de bolinha (disc, circle ou square)

  • Essa é uma lista NÃO ordenada
  • Com bolinhas abertas
