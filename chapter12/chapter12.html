Utilizando CSSs no código

Percebeu que essa página está com uma fonte diferente da que vimos nos últimos capítulos? Isso é porque eu estilizei ela com CSS!

As CSSs (folhas de estilo) têm o objetivo de estilizar os elementos da página com cores e tamanhos diferentes, dentre tantas outras funções. Podemos utiliza-las de 3 formas diferentes: inline, interna e externa.

Usando CSS inline

Para trazer uma característica de estilo a um parágrafo ou título, basta utilizarmos o parâmetro style dentro da tag que desejamos estilizar. Por exemplo:

<p style="color: red;background-color: yellow">Este parágrafo está com as letras em vermelho e com o fundo amarelo!</p>

Através desse código temos o seguinte resultado:

Este parágrafo está com as letras em vermelho e com o fundo amarelo!

Aplicar a estilização de forma inline não é recomendada devido ao fato de ser muito limitada. Você só consegue estilizar uma tag de cada vez e isso complica quando precisa corrigir várias tags com o mesmo estilo.
